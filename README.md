# How To Use?

Run `node script/build.js` or `npm run build` to build the project into 'build' directory.

Run `node script/start.js` or `npm run start` to preview the behavior of the project.

---

# 如何使用？

运行 `node script/build.js` 或 `npm run build` 来把项目构建至 build 文件夹。

运行 `node script/start.js` 或 `npm run start` 来预览项目的表现。
